Squarespace Integration

Embed the interactive Body Oracle directly into any Squarespace page using one of the methods below. The embed version loads without the site header and footer, fitting cleanly inside your Squarespace layout.

1 Quick Embed — Code Block (Recommended)

This is the simplest approach. It loads the Body Oracle inside an iframe on your Squarespace page.

  1. In your Squarespace editor, click + Add Block → Code
  2. Paste the embed code below into the code block
  3. Make sure the "Display Source" toggle is OFF
  4. Click Save and preview your page
<iframe src="https://sherlockfit.github.io/index.html/embed.html" width="100%" height="900" style="border:none; border-radius:8px; max-width:1400px;" title="SherlockFit Body Oracle" loading="lazy" allow="clipboard-write"> </iframe>
Tip: Adjust the height="900" value to fit your page design. For mobile-friendly layouts, try height="1200".

2 Deep Link to a Specific Zone

You can link directly to a specific body zone using the ?zone= parameter. This is great for blog posts or pages about a specific body area.

<iframe src="https://sherlockfit.github.io/index.html/embed.html?zone=heart" width="100%" height="900" style="border:none; border-radius:8px; max-width:1400px;" title="SherlockFit Body Oracle — Heart" loading="lazy" allow="clipboard-write"> </iframe>

Available zone names for deep linking:

crown, thirdEye, eyes, ears, nose, jaw, throat, neck, shoulders, collarbone, chest, heart, ribcage, solarPlexus, back, spine, belly, sacral, pelvis, arms, elbows, wrists, hands, root, hips, thighs, legs, knees, ankles, feet, lungs, stomach, liver, kidneys, intestines, skin

3 Simple Link Button

If you prefer to link visitors to the full Body Oracle page instead of embedding, add a button in Squarespace that links to:

https://sherlockfit.github.io/index.html/

In Squarespace: Add Block → Button → set the URL above as the link destination and choose "Open in new tab".

4 Squarespace Code Injection (Site-wide)

To add the Body Oracle to your site header or footer across all pages:

  1. Go to Settings → Advanced → Code Injection
  2. Paste the iframe code from Step 1 into the Footer field
  3. Click Save
Tip: The embed page works on any website platform that supports HTML — not just Squarespace. Use the same iframe code on WordPress, Wix, or any custom site.
