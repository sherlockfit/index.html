Squarespace Members Area — Copy This Code

You created a free Members Area on Squarespace — great choice! Members must sign up to access the page, which protects your Body Oracle and captures their name & email. Here's exactly what to paste:

1 Open your Members Area page in the Squarespace editor

Go to your Squarespace dashboard → Member Areas → open the page you created for the Body Oracle.

2 Add a Code Block

  1. Click + Add Block (or Add Section → Blank Section)
  2. Choose Code from the block menu

3 Paste this code and click Save

Copy the code below and paste it into the Code block. Make sure the "Display Source" toggle is OFF.

<div style="text-align:center; padding:1rem 0;"> <h2 style="font-family:Georgia,serif; font-size:1.5rem; color:#3a2f2b; margin:0 0 0.25rem;">Vitruvian Man Body Oracle</h2> <p style="font-size:0.8rem; letter-spacing:0.2em; color:#4a90d9; margin:0 0 1rem;">SHERLOCKFIT · BEYOND ELEMENTARY</p> </div> <iframe src="https://sherlockfit.github.io/index.html/embed.html" width="100%" height="900" style="border:none; border-radius:8px; max-width:1400px; display:block; margin:0 auto;" title="SherlockFit Body Oracle" loading="lazy" allow="clipboard-write"> </iframe> <p style="text-align:center; font-size:0.75rem; color:#999; margin-top:0.5rem;"> Click any body zone on the map to begin your investigation. </p>

4 Preview & publish

  1. Click Save on the code block
  2. Preview the page — you should see the interactive Body Oracle map

Tip: Because this page is inside your Members Area, visitors must sign up (free) or log in before they can see it. Squarespace automatically captures their name and email — no extra forms needed!
Mobile tip: If the map looks cramped on phones, change height="900" to height="1200" in the code above.

That's it! Your members will see the full interactive Body Oracle after signing up.

Other Integration Options

Not using a Members Area? You can also embed the Body Oracle on any regular Squarespace page, or link to it directly.

A Quick Embed — Any Page

Add a Code block to any Squarespace page and paste this:

<iframe src="https://sherlockfit.github.io/index.html/embed.html" width="100%" height="900" style="border:none; border-radius:8px; max-width:1400px;" title="SherlockFit Body Oracle" loading="lazy" allow="clipboard-write"> </iframe>

B Deep Link to a Specific Zone

Open the map directly to a body zone — great for blog posts. Add ?zone=heart (or any zone name) to the URL:

<iframe src="https://sherlockfit.github.io/index.html/embed.html?zone=heart" width="100%" height="900" style="border:none; border-radius:8px; max-width:1400px;" title="SherlockFit Body Oracle — Heart" loading="lazy" allow="clipboard-write"> </iframe>

Available zone names:

crown, thirdEye, eyes, ears, nose, jaw, throat, neck, shoulders, collarbone, chest, heart, ribcage, solarPlexus, back, spine, belly, sacral, pelvis, arms, elbows, wrists, hands, root, hips, thighs, legs, knees, ankles, feet, lungs, stomach, liver, kidneys, intestines, skin

C Simple Link Button

Link visitors to the full standalone page: Add Block → Button → use URL:

https://sherlockfit.github.io/index.html/

D Code Injection (Site-wide)

Add the Body Oracle across all pages: Settings → Advanced → Code Injection → paste the iframe code into the Footer field.

Tip: The embed works on any platform — Squarespace, WordPress, Wix, or any custom HTML site.
